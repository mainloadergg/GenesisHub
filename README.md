## 🎯 Overview

Genesis Hub is a powerful, feature-rich automation script for **King Legacy** on Roblox. Built with the GenesisX UI Library, it offers seamless auto-farming, sea event hunting, raid boss detection, smart server hopping, and much more — all with a modern, customizable interface.

> ⚠️ **Disclaimer**: This script is for educational purposes only. Use at your own risk. The developers are not responsible for any bans or penalties.

## ✨ Features

### 🌊 Sea Events System
| Feature | Description |
|---------|-------------|
| **Auto Serpent** | Automatically hunts Serpent in all seas |
| **Auto Sea King & Hydra** | Hunts Sea King and Hydra in Sea 2 with live timer display |
| **Auto Ghost Ship** | Detects and loots Ghost Ship chests automatically |
| **Auto Set-Sail (Sea Events)** | Hunts Drakenfyr, Abyssal Tyrant, Deepsea Crusher in Sea 3 |
| **Auto Galleon** | Hunts Galleon bosses in Sea 3 |
| **Auto Race Island** | Teleports to active Race Islands automatically |
| **Live Spawn Timers** | Real-time countdown for Sea King, Hydra, Ghost Ship, and Third Sea events |

### ⚔️ Combat & Farm
| Feature | Description |
|---------|-------------|
| **Auto Farm Level** | Automatically farms mobs based on your current level |
| **Auto Farm Material** | Farms selected materials with quest integration |
| **Auto Farm Boss** | Hunts specific bosses with dropdown selection |
| **Auto Farm All Bosses** | Kills all available bosses in the server |
| **Auto Kill Nearby Mobs** | Kills mobs within configurable distance |
| **Auto Bounty (Legit)** | Rotates through all sea bosses for bounty |
| **Auto Daily Quest** | Completes daily quests automatically (Sea 3) |

### 🏴‍☠️ Raid Bosses

| **Expert Swordman** | Sea 1 exclusive raid boss |

| **Spawn Status Panel** | Live tracking of which raid bosses are currently spawned |

### 🎣 Fishing System
| Feature | Description |
|---------|-------------|
| **Auto Fish** | Perfect catch automation with position saving |
| **Serpent Whirlpool Mode** | Auto-fishes in Serpent whirlpool when available |
| **Fishing When Not Spawned** | Falls back to saved position when no events are active |

### 🧬 Character & Stats
| Feature | Description |
|---------|-------------|
| **Auto Stats Distribution** | Automatically distributes stat points (Melee, Defense, Sword, Fruit) |
| **Auto Haki** | Auto-activates Buso (Armament), Ken (Observation), and Hao (Conqueror's) |
| **Auto Race V3** | Activates race ability at configurable HP threshold |
| **Walk On Water** | Allows walking on sea surface |
| **V Method (Zoan & Ope)** | Smart transformation handling for Ope-Ope fruit |
| **Legacy Pose** | Unlocks legacy pose animations |
| **Custom Animations** | Sets custom idle animations (Annoy, Egoistic, Protagonist, etc.) |

### 🛒 Shop & Items

| **Random Fruit** | Opens random fruits with selected keys (x1 or x10) |
| **Auto Buy Materials** | Purchases selected materials from Material Dealer |
| **Style Shop** | One-click purchase for all fighting styles |
| **Auto Store Fruits** | Automatically stores fruits in inventory |
| **Bring Fruits** | Teleports spawned fruits to player |

## 🚀 Quick Start

```lua
-- Load Genesis Hub
local GenesisX = loadstring(game:HttpGet("https://raw.githubusercontent.com/mainloadergg/Library/refs/heads/main/GenesisX.lua"))()

-- The script will automatically:
-- 1. Create the UI window
-- 2. Load your saved settings
-- 3. Start all enabled automation loops
```
